# Students Table Assignment

This project is a Student Management Dashboard built as part of a Full Stack Developer assignment.

The application allows users to manage student records through a clean and responsive interface.

## Features

- Add Student
- Edit Student
- Delete Student
- Form Validation
- Search / Filter Students
- Simulated Loading State
- Download student data as Excel-compatible CSV
- Responsive UI design

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js (optional backend)

## How to Run

### Option 1 (Recommended)

## Run Backend
cd backend
npm install
npm start

## Run Frontend
cd frontend
npm install
npm start

Backend: http://localhost:5000
Frontend: http://localhost:3000

## Project Purpose
This project demonstrates basic CRUD operations, frontend validation, and simple UI design for managing student data.
